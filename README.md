# Blackhole-simulation-c++
Simulasi efek lensa gravitasi dan ray tracer lubang hitam menggunakan C++ tanpa dependency eksternal.

Proyek Ray Tracer 3D ringan yang dibuat menggunakan C++ tanpa library tambahan atau game engine untuk mengsimulasikan efek fisika Teori Relativitas Umum disekitar lubang hitam, khususnya Gravitational Lensing (pembelokan cahaya karena gravitasi yang ekstrem).

Fitur:
- C++ Murni (tanpa dependency tambahan)
- Simulasi Gravitational Lensing (Memvisualisasikan bagaimana gravitasi extreme dari lubang hitam melengkungkan ruang-waktu dan membelokkan cahaya)
- Optimasi Performa Kompliasi
- Format gambar metah (.ppm)
